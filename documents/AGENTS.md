# Documents Guide

This folder contains paper, report, slide, screenshot, and generated document assets.

Treat most files here as deliverables or source assets for deliverables, not runtime code. `slides/` contains JavaScript slide-generation sources; `assets/` and screenshot folders are often generated or manually curated.

When working on document artifacts, preserve formatting intent and verify rendered output when possible. Avoid sweeping changes to generated media unless the task is explicitly about regenerating the document set.
